Github Instructions to upload

Firstly, make sure I'm in repository folder

  • git status (identifies the changes)
  • git add -A (adds all the files)
  • git commit -m "any message" (Commit is a snapshot(version) of the project and -m adds a message to push with the update)
  • git push -u origin main (Pushes the commit with '-u' meaning upstream(uploading) and origin main specifies the branch its being uploaded to)
